Face Robot

Ongoing

November 2025 - Present

Overview

A 22-DOF (Degrees of Freedom) facial expression robot that combines AI-driven animation with realistic mechanical expression capabilities. This project integrates NVIDIA's Audio2Face technology with custom robotics to create lifelike facial animations synchronized with speech.

Key Features

  • Integrating chat-based AI with NVIDIA Audio2Face model for realistic facial animation
  • Real-time voice detection with emotion recognition for responsive interactions
  • 22 independent degrees of freedom for nuanced facial expressions
  • Designing silicone skin in Blender with custom texturing and materials
  • 3D printing molds for precise silicone casting of facial components
  • Custom servo control system for smooth, natural movements
