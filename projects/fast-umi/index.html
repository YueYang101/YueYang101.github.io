Fast UMI for Dual Robot Arms

Ongoing

November 2025 - Present

Overview

A machine learning-based manipulation system that uses Universal Manipulation Interface (UMI) principles to enable efficient learning from human demonstrations. The system captures precise pose and spatial information during demonstrations for training dual robot arms.

Key Features

  • Building UMI system with identical grippers for consistent demonstration capture
  • Integrated ultra-wide camera for comprehensive scene understanding
  • Motion sensors for precise trajectory tracking during demonstrations
  • Capture pose and spatial information during human demonstrations
  • SLAM integration for robust localization and mapping
  • Machine learning pipeline for imitation learning
  • Dual-arm coordination for complex manipulation tasks
