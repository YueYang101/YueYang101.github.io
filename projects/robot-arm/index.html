Robot Arm Simulation

Completed

July 2025 - October 2025

Overview

A complete simulation and control system for the Lerobot SO101 robotic arm, featuring reinforcement learning training in NVIDIA Isaac Lab and intuitive teleoperation using a PS5 controller for data collection and demonstration purposes.

Key Features

  • Replicated Lerobot SO101 follower arm in simulation environment
  • Built reinforcement learning pipeline using NVIDIA Isaac Lab
  • Physics-accurate simulation for realistic training scenarios
  • Developed teleoperation interface using PS5 controller
  • Intuitive control mapping for human demonstration collection
  • Sim-to-real transfer capabilities for physical deployment
  • Modular architecture for easy extension and modification
